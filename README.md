# 🌤 Telegram Weather Bot

## 📌 Описание проекта
Этот проект представляет собой Telegram-бота, который показывает текущую погоду в любом городе с использованием OpenWeather API.

---

## 🎯 Цель проекта
Создать простое приложение, которое получает данные через API и выводит их пользователю в Telegram.

---

## ⚙️ Функционал
- /start — приветствие и инструкция
- /weather — получение погоды по городу
- /cityhelp — список доступных городов и помощь

---

## 🌐 Используемые технологии
- Python
- Telegram Bot API
- OpenWeather API
- requests library

---

## 📡 Как работает бот
1. Пользователь вводит команду /weather город
2. Бот отправляет запрос к OpenWeather API
3. Получает данные о погоде
4. Отправляет результат пользователю

---

## 🚀 Пример использования
/weather Almaty  
/weather Oskemen
